You’re starting a request to get permissions for yourself to use NDSP. The permission you receive will depend on the role available for the collection and, if more than one role is offered, the one you select.

You can use this process whether you’re new to National Data Submission Portal or already have an account and need additional access, for example, to a different data collection or organisation. Each request covers one organisation and one data collection, and you can request access for your own organisation or another organisation you submit data for.

You can read more about the available roles in the permission guidance (opens in a new tab).

What you’ll need to provide

Most of the requests require you to confirm the organisation’s ODS code.
If you don’t know the organisation’s ODS code, you can find it on ODS portal (opens in a new tab).

If you need Information Governance (IG) manager's approval, you’ll also be asked to provide the name and contact details of your organisation’s IG manager.
