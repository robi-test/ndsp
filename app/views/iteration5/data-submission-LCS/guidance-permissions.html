If you need help using the National Data Submission Portal (NDSP), you can find answers and guidance using the resources below.

NDSP guidance

You can visit our NDSP guidance for step-by-step instructions and information about:

  • Accessing the portal
  • Managing your permissions and roles
  • How to submit data
  • Setting up API submissions

Lung cancer screening dataset (LCSDS) resources

Quick access to guidance and tools to help you manage your data submissions.

  • LCSDS implemention tools: Implementation tools (this includes ETOS and other resources).
  • LCSDS timetable: Full list of current submission periods and deadlines.
  • LCSDSß schemas: Access the latest schema files (this includes XML schema and CSV file).
How to start setting up an API submission

Your organisation can use the API to automate submissions. This is a one-off setup and is only needed if submissions are made programmatically. See the API submission guidance for full instructions.

How to get an API token

Click Get an API token above to open the API management application.
Go to OAuth & permissions and select Rotate secret to generate a new client secret. This client secret is your API token.
Keep your API token secure, as it allows submissions on behalf of your organisation.

Other settings

When submitting via the API, you must also provide a submission type and a submission window ID.
