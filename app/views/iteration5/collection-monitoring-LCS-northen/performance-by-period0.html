Performance by period

More about the data on this page

Reporting period
The period when the activity happened.

Submission window status
Indicates whether the submission window is currently Open or Closed, this based on the Submission window end date.

Total file submitted
The total number of files submitted during the submission window, including successful, rejected, and resubmitted files (excluding test files).

Organisations with successful submission
The number of organisations that had at least one file successfully accepted during the submission window (excluding test files).

Organisations not submitted
The number of organisations that did not have any files successfully accepted during the submission window (excluding test files).

Number of resubmissions
The number of files resubmitted during the submission window, counted only for valid submissions (excluding test files).

File rejection (%)
The number and percentage of submitted files that were rejected during validation checks (excluding test files).

Record rejection
The number of records that were rejected during validation checks (excluding test files). This includes records in files that have been rejected.
