How to start setting up an API submission

Your organisation can use the API to automate submissions. This is a one-off setup and is only needed if submissions are made programmatically. See the API submission guidance for full instructions.

How to get an API token

Click Get an API token above to open the API management application.
Go to OAuth & permissions and select Rotate secret to generate a new client secret. This client secret is your API token.
Keep your API token secure, as it allows submissions on behalf of your organisation.
