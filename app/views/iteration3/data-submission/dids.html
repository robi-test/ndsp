Automating submissions with the API

Your organisation can use the API to automate submissions instead of uploading files manually. This requires a one-off setup using an API token and is only needed if submissions are made programmatically. See the API submission guidance for full setup instructions and examples.

To get an API token, click the 'Get an API token' button above. This will take you to the application used to manage API access. From there, go to the 'OAuth & permissions' tab and select 'Rotate secret' to generate a new client secret. This client secret is your API token.

Keep your API token secure and do not share it, as it provides access to submit data on behalf of your organisation.
