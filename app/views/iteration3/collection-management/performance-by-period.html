Performance by period

More information about this table

Reporting period
The reporting period that the submissions relate to.

Submission window end date
The date when the submission window closes for that period.

Submission window status
Indicates whether the submission window is currently Open or Closed, this based on the Submission window end date.

Total submissions
The total number of files submitted during the submission window, including successful, rejected, and resubmitted files (excluding test files).

Providers submitted
The number of providers that submitted at least one file during the submission window (excluding test files).

Providers with successful submission
The number of providers that had at least one file successfully accepted during the submission window (excluding test files).

Number of resubmissions
The number of files resubmitted during the submission window, counted only for valid submissions (excluding test files).

File rejection (%)
The percentage of submitted files that were rejected during validation checks (excluding test files).

Test submissions
The number of submissions identified as test file type.
